Jest OpenTelemetry syntax is simple and easy to use.
Each test should begin by sending a request to any of your backends in your test environments, like this:

<Note>
  Note the use of the `axiosInstance` returned from `traceloop` to send the
  request.
</Note>

```js
const traceloop = new TraceLoop();

await traceloop.axiosInstance.post('http://my.awesome.website/orders/create');
await traceloop.fetchTraces();
```

Following that, you can run assertions on anything that happened in your system as part of this request.
